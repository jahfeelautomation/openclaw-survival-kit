# OpenClaw Survival Kit — Workspace Context

**What this is:** Open-source toolkit of watchdogs and self-healing scripts for AI agent systems. Published on GitHub. Our community/developer outreach product.

**What's in here:**
- `claw-drift/` — Configuration drift detection tool
- `README.md` — Public-facing documentation
- `TEST.md` — Test documentation

**Current state:** Promoted and built (Batches 15-16 complete). Core tools functional. Published to GitHub.

**Process:**
1. Build new tools as standalone directories (claw-*)
2. Each tool gets its own README and test
3. Git push requires JahFeel approval
4. Community feedback drives priorities

**Staging:** `openclaw-survival-kit-staging/` is the sibling directory with additional tools in development (claw-channel-watch, claw-cron, claw-medic, claw-pin, claw-reaper, claw-session-repair).

**Skills needed:** PowerShell, Python, cross-platform scripting
